CRITICAL
EXPOSED ADMIN PANEL — NO AUTH
Your admin dashboard at /admin is publicly accessible without authentication. Anyone on the internet can access it and view customer data.
CRITICAL
SQL INJECTION IN LOGIN FORM
The login endpoint is vulnerable to SQL injection. An attacker can bypass authentication or dump your entire database in minutes.
HIGH
OUTDATED TLS — ACCEPTS TLS 1.0
Your server accepts TLS 1.0 connections which have known vulnerabilities. Should only accept TLS 1.2 and 1.3.
HIGH
API KEYS EXPOSED IN JS BUNDLE
A live AWS API key was found hardcoded in your frontend JavaScript. Anyone viewing your source can use it to access your cloud account.
MEDIUM
MISSING SECURITY HEADERS
X-Frame-Options, CSP, and HSTS headers are absent. This enables clickjacking and other browser-based attacks against your users.
